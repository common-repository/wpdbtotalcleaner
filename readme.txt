WPDBTotalCleaner is a very simple, multisite compatible tool to clean Wordpress Database.

== Description ==

WPDBTotalCleaner is a very simple tool which can dramatically reduce your Wordpress database.
This plugin is based on WPCommentCleaner, developed by WordpressLover.com.

== Installation ==

1. Upload `wpdbtotalcleaner` folder to the `/wp-content/plugins/` directory
2. Activate WPDBTotalCleaner through the 'Plugins' menu in WordPress
3. Go To The Plugin Admin Page Under WPDBTotalCleaner Menu
